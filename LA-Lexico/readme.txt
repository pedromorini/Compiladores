#Analisador Léxico desenvolvido para a linguegem LA

*Pedro Morini da Mota - 795414

-Desenvolvimento-

O trabalho foi desenvolvido com as seguintes ferramentas no sitema operacional Windows 10

-Java 17 - https://www.oracle.com/br/java/technologies/downloads/#java17
-Apache Maven - (Selecione o Plugin Apache Maven no momento de instalçao do Netbeans)
-Antlr 4.7.2 - https://www.antlr.org/download.html
-Netebeans IDE 17 - https://netbeans.apache.org/download/index.html
-Corretor automático - https://github.com/dlucredio/compiladores-corretor-automatico

-Compilação-

Com as ferramentas instaladas, abra o projeto no Netbeans clica com o botão direito no ícone do projeto e selecione build
Será gerada uma pasta target no projeto e o arquivo LA-Lexico-1.0-SNAPSHOT-jar-with-dependencies dentro dessa pasta para executar o programa

-Execução-

No prompt de comando utilize o comando 
java -jar Caminho/Ate/O/Arquivo/Compilado/LA-Lexico-1.0-SNAPSHOT-jar-with-dependencies.jat Caminho/Ate/O/Arquivo/De/Entrada/entrada.txt Caminho/Ate/O/Arquivo/De/Saida/saida.txt 
para executar o programa.
